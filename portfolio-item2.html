Project Overview

Pollinate is a polling application. I wanted to create something clean looking with a bee theme as my nickname is Bee. Bees pollinate and spread flowers around, so it was fitting to call it pollinate to represent polls you want to spread around!

It started quite simple but over time I have gathered feedback from users and added additional features / tweaked certain elements.

Here are the list of changes I made to improve user experience:

  • fixed an issue where the max vote was off by 1
  • made app more mobile friendly
  • added visual bars to display data
  • added text to bars
  • added ability to copy link and added copy icon
  • added remove button
  • pressing enter now submits the form
