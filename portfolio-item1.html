Project Overview

Busy Bee is an E-Commerce website for pet accessories. I created this website to practice my frontend skills using React and backend skills using Strapi and Stripe.

It features a functional image carousel, a main store item page with categories and price filters, item detail pages for each store item, a dynamic shopping cart with the ability to add and remove items, and checkout page with payment.
