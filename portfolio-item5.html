Project Overview

Robeeto is a web app that makes your game of Uno more interesting. I created this web app after playing Uno on holiday and wishing there was a way to make it more random and exciting. .

It features player number and name inputs (2-10), many different randomised events, score tracking and game round tracking.

Screen to enter the number of players, supporting up to 10 players following the standard rules for Uno.

Screen to enter the player's names. Number of boxes corresponds to the number of players there are.

Main page featuring the game round tracking which increases by one everytime scores are tracked. Play card button which is pressed everytime a card is played. Reset all button which takes you back to the screen to enter the number of players. Track winner button to track scores when someone has won.

Fully voiced event page which tells you the type of event triggered, the players involved, the instruction for the event, and the player that will continue play.

Score tracking for every round and it keeps track of the total score for each player.

Reveal the overall winner after multiple games and see who ends up on the podium!
